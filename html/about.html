Corbli is a personal project created by Anthony.

I started programming in my last semester of University after I decided to take an introduction to programming class that focussed on C++. I have been playing with code ever since, but although I'd heard a lot about it, I never considered web development to be "real programming". Making pretty pages with a small amount of javascript seemed like more of a design job than a coding one. Thus I regularly dismissed the idea of any web development project. Nonetheless however, web development is booming and has been for some time. As hard as I tried to ignore it, ultimately I couldn't. So I decided to to try my hand at building my first website.

First, I knew I would need a domain name. I also knew that I needed to buy it from a registrar. Initially I planned to use GoDaddy, but was warned not to by the people of the internet. So I found my way to Namecheap and started searching for a nice domain name for myself. Inititally I was planning to use something along the lines of anthonysurname.com, which was available.
